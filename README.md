# heart_disease_prediction_kaggle_project
# ❤️ Heart Disease Prediction

## 📌 Problem
Predict whether a patient has heart disease based on medical features.

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- LightGBM

## 🔍 Approach
- Data preprocessing (handling categorical values)
- Train-test split with stratification
- Model: LightGBM Classifier

## 📊 Results
- Achieved strong accuracy on test data
- Model captures important health indicators

## 🚀 How to Run
```bash
pip install -r requirements.txt
python src/evaluate.py
